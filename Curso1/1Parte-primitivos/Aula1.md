Outros tipos de dados:
- null: ausência de valor. Pode ser atribuído como valor de uma variável. Quando uma variável existe e é iniciada.
- undefined: ausência de valor, a variável não existe ou não foi inicializada. Considerada ausência de valor inesperada.


// Convenções do código
- JS é Case sensitive

- camelCase: Inicia com letra minúscula e a letra de cada palavra em seguida é escrita com letra maiúscula.
minhaVar
senhaDoUsuario

- snake_case: espaços são substituidos por _, todas palavras com letras minúsculas

- PascalCase: todas as palavras começam com letras maiúsculas
